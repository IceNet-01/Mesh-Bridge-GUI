# Screenshots Directory

Place your application screenshots here with the following naming:

- `dashboard.png` - Dashboard view showing radios, messages, and statistics
- `messages.png` - Message monitor with live feed
- `network-health.png` - Network health metrics and analysis
- `emergency.png` - Emergency response and weather alerts
- `map.png` - Interactive map view with nodes
- `ai-settings.png` - AI assistant configuration
- `tactical.png` - TAK tactical view

## Taking Screenshots

1. Run the application: `npm run start`
2. Open browser to http://localhost:5173
3. Navigate to each page
4. Take screenshots (use browser tools or screenshot utility)
5. Save to this directory with the names listed above
6. Screenshots will automatically appear in the README

## Image Guidelines

- Format: PNG preferred (smaller file size for GitHub)
- Resolution: 1920x1080 or similar (will be scaled in README)
- Compress images to reduce repository size
- Show realistic data (blur sensitive information if needed)
